Dotfiles Manual Extension
=========================

Installation
------------

Install this extension using the [`dotfiles`](https://github.com/stefaniuk/dotfiles) command.

```bash
$ ./dotfiles --extension-man
$ reload
```

Usage
-----

```bash
$ dm --help

Usage: dotfiles-man term1 [term2 term3 ...]

Options:
        -a|--list-all
        -r|--show-raw
        -h|--help
```
